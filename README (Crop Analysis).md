# 🌾 Crop Recommendation Analysis

## 📌 Project Overview
This project analyzes agricultural data to recommend suitable crops based on soil nutrients and environmental conditions.

The dataset includes features like Nitrogen (N), Phosphorus (P), Potassium (K), temperature, humidity, pH, and rainfall.

---

## 🎯 Objective
To understand how different factors affect crop growth and extract meaningful insights using data analysis and visualization.

---

## 📊 Dataset Features

- **N** → Nitrogen content in soil  
- **P** → Phosphorus content  
- **K** → Potassium content  
- **temperature** → Temperature in °C  
- **humidity** → Humidity percentage  
- **ph** → Soil pH value  
- **rainfall** → Rainfall in mm  
- **label** → Crop name (target variable)

---

## 🛠️ Technologies Used

- Python 🐍  
- Pandas 📊  
- NumPy 🔢  
- Matplotlib 📉  
- Seaborn 🎨  

---

## 🔍 Analysis Performed

- Data Cleaning & Exploration  
- Statistical Summary using `describe()`  
- Feature Engineering:
  - Total Nutrients (N + P + K)
  - pH Type (Acidic, Neutral, Alkaline)
  - Rainfall Category  
- GroupBy Analysis  
- Correlation Analysis using Heatmap  
- Visualization:
  - Scatter Plot  
  - Box Plot  
  - Heatmap  

---

## 📈 Key Insights

- Crops require different nutrient combinations  
- Rainfall and humidity play a major role in crop selection  
- Most features are weakly correlated → each factor is important  
- Neutral pH (6.5–7.5) is suitable for many crops  

---

## 🌱 Practical Use Case

This analysis can help:
- Farmers choose the right crop  
- Improve crop yield  
- Make data-driven agricultural decisions  

---

## 🏁 Conclusion

Crop production depends on multiple environmental and soil factors.  
By analyzing these features, better farming decisions can be made.
